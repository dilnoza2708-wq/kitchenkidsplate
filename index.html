⚙️ Подключение Supabase (облачная БД)

Для работы в реальном времени нужно создать бесплатный проект на supabase.com. Данные будут синхронизироваться у всех сотрудников автоматически.

  1. Зайдите на supabase.com → New Project (бесплатно)
  2. После создания: Settings → API → скопируйте Project URL и anon key
  3. Перейдите в SQL Editor и выполните SQL из кнопки ниже
  4. Вставьте URL и ключ сюда и нажмите «Сохранить»
